For loope in js

In JavaScript, loops allow you to execute a block of code repeatedly.
There are several types of loops in JavaScript, each serving a specific purpose.
Here's an overview of the most common loops in JavaScript:

for loop
  • The `for` loop is the most commonly used loop. It repeats a block of code a certain number of times, and it consists of three parts: initialization, condition, and iteration.

    • for (let i = 0; i < 5; i++) {
    console.log(i);
    };

    while loop
  • while loop continues to execute as long as a specified condition evaluates to `true`.

    • let i=0;
    while (i < 5) {
    console.log(i);
    }

    do...while loop
  • The do...while loop is similar to the `while` loop,
    but the condition is checked **after** the loop body runs, meaning the loop will always run at least once.

    • let i=0;
    do { console.log(i);
    } while (i < 5);

    for...in loop
  • The for...in loop is used for iterating over the properties of an object.

    • const person={ name: "Alice" , age: 25, occupation: "developer" };
    for (let key in person) {
    console.log(key + ": " + person[key]);
    }

    for...of` loop
  • The for...of loop is used for iterating over **iterable objects** (like arrays, strings, maps, etc.).

    • const fruits=["apple", "banana" , "cherry" ];
    for (let fruit of fruits) {
    console.log(fruit);
    }

    forEach() (Array Method)
  • forEach() is a higher-order function available on arrays to iterate over the elements.

    • const numbers=[1, 2, 3, 4, 5];
    numbers.forEach((number)=> {
    console.log(number);
    });

    Example of Nested Loops
  • You can also use loops inside other loops, known as **nested loops**.

    • for (let i = 0; i < 3; i++) {
    for (let j=0; j < 3; j++) {
    console.log(`i: ${i}, j: ${j}`);
    } }

    Loop Control Statements - **`break`**
  • Exits the loop entirely.

    • for (let i=0; i < 5; i++) {
    if (i===3) {
    break;
    } console.log(i);
    }

    continue
  • Skips the current iteration and proceeds to the next iteration.

    • for (let i=0; i < 5; i++) { if (i===3) {
    continue;
    }
    console.log(i);
    }
